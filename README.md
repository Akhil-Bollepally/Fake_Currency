## Project Overview
The Fake Currency Detection System is a **Django-based web application** designed to detect whether a currency note is **genuine or fake** using **Machine Learning and image processing techniques**.

The system allows users to upload an image of a currency note through a web interface. The uploaded image is analyzed using a trained machine learning model, and the prediction result is displayed to the user.

This project is developed as a **simple and independent academic project** to demonstrate the practical application of **Machine Learning in real-world security problems**.

---
